# Goal

Your job is to make an AJAX request using fetch to this event API:
https://www.tvmaze.com/api

# Part 1 - Page setup

Your job is to make an AJAX request shows using this root url:

```text
https://api.tvmaze.com/
```

The available endpoints are in the [docs](https://www.tvmaze.com/api)

# Part 1 - Search

In a web page, allow users to search for shows and display up to 3 options.

## Requirements

The show search should include:

- Provide an input where users can enter a show name to find options.
- Require at least 3 characters in the show name input before making a search request.
- Show up to 3 show listings. If fewer shows are available, display those found.
- Notify the user if no shows were found.

Each show result should display:

- Show Name
- The Average Rating (if available)
- A medium sized image

# Part 2 - Details

Provide a detailed view for each show when selected.

## Requirements

When a show is selected, display a detailed view that includes:

- A larger sized image
- Show Name
- Detailed Summary (if available)
- Genres
- Language
- A list of all the episodes. If any episode is clicked, it should redirect you

## Tips

- Use the TVMaze API endpoint for job searches.
- Handle errors by notifying the user if something goes wrong with the API call
