## Table of Contents

Inserting a Table of Contents (TOC) in Microsoft Word is a straightforward process that can be done automatically based on your document's heading styles.

Step 1: Before inserting the TOC, ensure that youâ€™ve used Wordâ€™s built-in "Heading Styles" for your document headings.

# Templates

## Saving Templates

- Click on File in the top-left corner.
- Select `Save As`.
- Choose your `Templates` folder to Save your Template in
- In the Save as type dropdown (Windows) or File Format dropdown (Mac), select Word Template `\*.dotx`.

## Using the Template

- Let's create a new document in word.
- Select `New from Template`
- To see your custom templates, select `Personal` and you should see your `Lab Report Template - JAC 2024`
- Happy creating :^)
