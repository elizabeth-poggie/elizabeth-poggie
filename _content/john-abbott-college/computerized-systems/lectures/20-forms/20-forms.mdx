# Intro to Forms

Forms offer a less painful way of adding and modifying data to a table.

## Overview

In this section we will cover:

- Create a form from an existing table
- Deleting unnecessary fields from a form
- Adding new form elements such as buttons for form navigation

# Exercises 1

- Open the [practice database](https://media.gcflearnfree.org/content/56fd8065577fba179c8b6a33_03_31_2016/access2016_sampledatabase.accdb).
- Create a form from the Customers table.
- Delete the subform.
- Create a combo box. Add the following choices to the combo box:

```text
Raleigh
Durham
Hillsborough
Cary
Chapel Hill
Garner
Charlotte
```

- Store the value in the City field, and label the combo box NC City List.
- Switch to Form view, and click the drop-down arrow in the combo box you created. You should see a list of cities.
