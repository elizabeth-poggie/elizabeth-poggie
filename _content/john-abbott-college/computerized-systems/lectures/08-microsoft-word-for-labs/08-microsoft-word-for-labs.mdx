# Overview

## Workbooks

A workbook is the entire Excel file that can contain multiple sheets (or tabs). Each workbook can be saved independently.

## Worksheet (Tab)

Each workbook consists of one or more worksheets, often referred to as tabs. Each worksheet contains a grid of cells where data is entered.

## Cell

Cells are the basic units of a worksheet, organized into a grid of rows and columns. Each cell is identified by its cell reference, which combines the column letter and row number. (e.g. A1, B2, C3, etc)

## Rows & Columns

Rows run horizontally and are numbered (e.g. 1,2,3)
Columns run vertically and are labeled with letters (e.g. A, B, C)
